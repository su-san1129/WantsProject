# WantsProject
(仮称)WantsProject / AngularとSpringBootの復習がてらSPAのポートフォリオ作成

家庭内で経費申請を行うアプリ。  
楽〇清算の承認者が嫁になるイメージ。  
ほしいものを申請すると、許可、拒否ができる。  
欲しいものリストは、Amazonや楽天などから取得することができる。  

## 各ファイルの在処
wants-project-api => SpringBootファイル  
wants-project-web => Angularファイル

## 概要等
open-jdk: 11  
angular-cli: 9
 
**覚えたいもの** 
 - JWT認証(JSON Web Token)
 - SpringDataJDBC
 - Angularのライフサイクル
 - rxJSライブラリの利用方法
 - Dockerを用いたコンテナ管理
 - AWSでの運用
